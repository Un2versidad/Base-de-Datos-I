# Base de Datos I - Talleres y Parciales

Este repositorio contiene todos los talleres, parciales y material de apoyo para la asignatura **Base de Datos I**. Aquí encontrarás ejercicios prácticos de SQL, diseño de bases de datos, y evaluaciones que te ayudarán a dominar los conceptos fundamentales de las bases de datos relacionales.

## 📚 Contenido del Repositorio

### 🛠️ Talleres
Los talleres están organizados por temas y dificultad progresiva:

- **Taller 1: Fundamentos de SQL**
  - Consultas básicas SELECT
  - Filtrado con WHERE
  - Ordenamiento con ORDER BY
  
- **Taller 2: Consultas Avanzadas**
  - JOINs (INNER, LEFT, RIGHT, FULL)
  - Subconsultas
  - Funciones de agregación
  
- **Taller 3: Diseño de Base de Datos**
  - Modelo Entidad-Relación
  - Normalización
  - Implementación de tablas
  
- **Taller 4: Procedimientos y Funciones**
  - Stored Procedures
  - Triggers
  - Vistas

### 📝 Parciales
Evaluaciones teórico-prácticas que cubren:

- **Parcial 1: Fundamentos y SQL Básico**
  - Conceptos teóricos de BBDD
  - Consultas SELECT básicas
  - DDL y DML
  
- **Parcial 2: Consultas Avanzadas y Diseño**
  - JOINs complejos
  - Subconsultas anidadas
  - Diseño ER
  
- **Parcial Final: Integración Completa**
  - Diseño completo de BBDD
  - Optimización de consultas
  - Casos de uso reales

## 🚀 Cómo Usar Este Repositorio

### Prerrequisitos
- Sistema de gestión de base de datos (MySQL, PostgreSQL, SQL Server)
- Editor de código (VS Code, DBeaver, etc.)
- Conocimientos básicos de SQL

### Configuración Inicial
1. Clona el repositorio:
   ```bash
   git clone https://github.com/Un2versidad/Base-de-Datos-I.git
   ```

2. Navega al directorio:
   ```bash
   cd Base-de-Datos-I
   ```

3. Ejecuta los scripts de creación de base de datos de prueba:
   ```sql
   -- Ejecutar scripts en scripts-bd/creacion-tablas/
   ```

### Trabajando con los Talleres
1. Cada taller incluye:
   - 📄 **Enunciado**: Descripción detallada del ejercicio
   - 📄 **Datos de prueba**: Scripts SQL para crear datos de ejemplo
   - 📄 **Solución**: Respuestas comentadas (revisar después de intentar)
   
2. Sigue el orden secuencial de los talleres
3. Practica con los datos de ejemplo proporcionados
4. Compara tus soluciones con las propuestas

### Preparación para Parciales
1. Revisa el material de apoyo correspondiente
2. Practica los talleres relacionados
3. Estudia los ejemplos y diagramas ER
4. Resuelve parciales anteriores como práctica

- Modelos de datos
- Arquitectura de SGBD
- Independencia de datos
- Integridad referencial

- DDL (Data Definition Language)
